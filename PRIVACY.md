# Privacy

This public package intentionally excludes personal candidate data.

Do not commit:

- resumes or portfolios containing private information;
- names, phone numbers, personal email addresses or home addresses;
- candidate profiles, work authorization evidence or immigration documents;
- application history, interview notes or recruiter correspondence;
- cookies, browser storage, sessions, passwords, OTPs or API secrets.

The automated task reads only public recruitment pages and writes normalized public job metadata. It does not log in or submit applications.
